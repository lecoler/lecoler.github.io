参数 -u
将本地的master分支推送到origin主机，同时指定origin为默认主机，后面就可以不加任何参数使用git push，这叫做simple方式，默认只推送当前分支
此外，还有一种matching方式，会推送所有有对应的远程分支的本地分支。
Git 2.0版本之前，默认采用matching方法，现在改为默认采用simple方式
